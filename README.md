# Netflix-Style Resume Website

A modern, interactive resume website inspired by Netflix's sleek design and user interface. This project creates an engaging way to showcase your professional profile, experience, and projects.

## Features

- **Netflix-Inspired Design**: Dark theme with red accents, mimicking Netflix's signature look
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Elements**: Hover effects, smooth transitions, and animated components
- **Project Modals**: Detailed project information displayed in Netflix-style modals
- **Smooth Navigation**: Sticky navigation with smooth scrolling between sections
- **Professional Sections**:
  - Hero section with profile introduction
  - Current projects and achievements
  - Professional experience timeline
  - Skills showcase with categorized tags
  - Featured projects with detailed descriptions
  - Educational background
  - Contact information

## Sections Overview

### 1. Hero Section
- Large profile photo with hover effects
- Professional title and description
- Call-to-action buttons for viewing experience and downloading resume

### 2. Continue Watching (Current Projects)
- Showcases ongoing projects and recent achievements
- Progress bars for projects in development
- Achievement badges for completed certifications

### 3. Professional Experience
- Netflix-style cards displaying work history
- Company information, duration, and key achievements
- Visual hierarchy with company logos and role descriptions

### 4. My List (Skills)
- Categorized skill tags
- Interactive hover effects
- Organized by Programming Languages, Frameworks, Tools, and Specializations

### 5. Featured Projects
- Portfolio projects displayed as Netflix movie cards
- Click to view detailed project information in modals
- Technology stack indicators
- Links to live demos and source code

### 6. Educational Background
- Timeline-style education display
- Degree information with graduation years
- Academic achievements and honors

### 7. Contact Section
- Professional contact information
- Social media and professional network links
- Location and availability information

## Customization Guide

### Personal Information
Update the following in `index.html`:
- Replace "Your Full Name" with your actual name
- Update professional title and description
- Add your actual contact information
- Replace placeholder education details

### Profile Photo
- Replace `profile-photo.jpg` with your professional headshot
- Recommended size: 350x350 pixels, square format
- Ensure high quality and professional appearance

### Experience Section
Update the experience cards with your actual work history:
- Company names and positions
- Employment dates
- Key achievements and responsibilities
- Company logos (optional)

### Skills Section
Customize the skill categories and tags:
- Add your programming languages
- Include frameworks and libraries you use
- List tools and technologies you're proficient in
- Highlight your specializations

### Projects Section
Replace placeholder projects with your actual work:
- Project titles and descriptions
- Technology stacks used
- Project images or screenshots
- Links to live demos and GitHub repositories

### Color Customization
To change the color scheme, update these CSS variables:
- Primary red: `#e50914` (Netflix red)
- Background: `#141414` (Dark gray)
- Cards: `#1a1a1a` (Lighter dark gray)
- Text colors: Various shades of white and gray

## Setup Instructions

1. **Download the files** to your local machine
2. **Customize the content** with your personal information
3. **Add your profile photo** (replace placeholder)
4. **Update project information** with your actual projects
5. **Test the website** by opening `index.html` in a web browser
6. **Host the website** on platforms like:
   - GitHub Pages
   - Netlify
   - Vercel
   - Firebase Hosting

## Hosting Options

### GitHub Pages
1. Create a new GitHub repository
2. Upload all files to the repository
3. Enable GitHub Pages in repository settings
4. Your resume will be available at `username.github.io/repository-name`

### Netlify
1. Drag and drop your project folder to Netlify
2. Get an instant live URL
3. Optional: Connect to GitHub for continuous deployment

### Custom Domain
- Purchase a custom domain for a more professional look
- Example: `yourname.dev` or `yourname.com`
- Update hosting platform settings to use custom domain
